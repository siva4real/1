Financial Modeling

Python Analysis of Revenue Trends


import pandas as pd
import matplotlib.pyplot as plt
import numpy as np

# Load quarterly financial data
data = pd.DataFrame({
    'quarter': ['Q1', 'Q2', 'Q3', 'Q4'],
    'revenue': [10.2, 11.5, 11.8, 12.5],
    'expenses': [8.1, 9.2, 9.4, 9.7]
})

# Calculate profit margin
data['profit_margin'] = (data['revenue'] - data['expenses']) / data['revenue'] * 100

# Display results
print(data)
print(f"Average profit margin: {data['profit_margin'].mean():.1f}%")

# Create visualization
plt.figure(figsize=(10, 6))
plt.plot(data['quarter'], data['revenue'], marker='o', label='Revenue')
plt.plot(data['quarter'], data['expenses'], marker='s', label='Expenses')
plt.title('Q4 2025 Financial Performance')
plt.xlabel('Quarter')
plt.ylabel('Amount ($B)')
plt.legend()
plt.grid(True)
plt.show()
For our more technical stakeholders, here's a Python script we used for trend analysis. This helps us visualize performance and make data-driven decisions.

Financial Formulas

Return on Equity (ROE)

$$ROE = \frac{Net\ Income}{Shareholder's\ Equity}$$

Our ROE for Q4: $$\frac{2.8B}{45.2B} = 6.2\%$$

Earnings Per Share (EPS)

$$EPS = \frac{Net\ Income\ -\ Preferred\ Dividends}{Weighted\ Average\ Shares\ Outstanding}$$

Q4 EPS: $$\frac{2.8B}{658M} = \$4.25$$

Debt-to-Equity Ratio

$$Debt-to-Equity = \frac{Total\ Debt}{Total\ Equity}$$
